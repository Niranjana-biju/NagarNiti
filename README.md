# NagarNiti 

AI-powered civic issue reporting and accountability platform.

## Features
- AI-powered issue detection from images
- Smart issue prioritization
- Complaint generation
- Google Maps integration
- Ward-wise dashboard
- Hotspot detection
- Escalation system

## Tech Stack
- React + Vite
- Tailwind CSS
- Firebase Firestore
- Groq API (Llama)
- Google Maps JavaScript API
